Выпуск №114

21 апреля 2017

Негодования абзац. Зачем-то вы захотели пописать для Windows 10 Mobile? Качаете инсталлятор VS, прорываетесь сквозь огромное количество доступных компонентов: .Net и на C++ отдельно, разработка игр и приложений для мобильных отдельно. Всё это весит конские гигабайты. Вот уж в 2017 жаловаться на размер, но там просто дофига. Думаете, скачали и можете запустить на эмуляторе? Ха. Качайте отдельно и попробуйте понять, чем отличаются эмуляторы из списка. Build Number разный? Что? У вас Windows в виртуалке? Что ж, вам нужна поддержка вложенных виртуальных машин, потому как иначе эмулятор не сможет запуститься. У вас Windows 10 через Bootcamp. Поздравляю, есть проблемы с драйверами беспроводной клавиатуры. Microsoft, а слабо уже вывести свою Visual Studio for Mac в такую кондицию, чтобы можно было открыть её и писать код, в том числе для W10M? Драйвера — не проблема MS. Выбор опций — так это ж чтоб только нужное. Вложенные виртуалки — так эмулятор, не симулятор какой-то. Всё круто, всё логично, только хрен код напишешь.
