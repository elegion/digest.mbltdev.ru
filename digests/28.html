WinObjC

В аккаунте Microsoft на GitHub появился репозиторий WinObjC. WinObjC позволит настроить окружение и запустить iOS-проект с помощью Visual Studio 2015 под Windows 10. Мостики сделаны не для всего. Например, поддержки Storyboard нет, но принятно видеть централизованную инициативу такого рода. Читайте Wiki, есть картинки.

github.com

spacetime

Экспериментальная библиотека от Facebook для изменения части слоев в реальном времени. Основана на малоизвестных составляющих Core Animation: CAReplicatorLayer и CAMediaTiming. Доступна в виде пода, распространяется под BSD.
