Выпуск №121

09 июня 2017

Астрологи объявили неделю WWDC. Количество твитов по теме возросло. Ещё бы, есть о чём рассказать. Слёзы радости: рефакторинг Swift в Xcode 9. Rename выглядит круто. Всё такое интерактивненькое. Появилась возможность запускать несколько симуляторов и делать скриншоты из XCUITest. Какой удар по fastlane! Подсказка: слабый. Шапки нашего дайджеста не хватит, да и вы, понятно, из всех утюгов слышите новости с WWDC. Но так-то радостно на душе, весело на душе. P.S. Вот только модульного компьютера не хватает, чтобы можно было по старинке память ставить, диски. Можно и видеокарту. Наверное, для бизнеса не алё. Наверное, не Apple Way ни разу. Но хочется.

Новости

HomeKit Accessory Protocol Specification

Раньше MFi была закрытой программой. Хочешь делать что-то железное для Apple и нужна спецификация? Становись частью MFi. Теперь это не так. Спецификация открыта для некоммерческого использования.

developer.apple.com

New rules following WWDC 2017

Обновились App Store Review Guidelines. По ссылке вы найдете красивый diff. Из интересного — в некоторых случаях можно загружать и выполнять код, не стоит использовать кастомные штуки для оценки приложения, нужно объяснять, зачем нужен доступ к специальным функциями (контакты, локация, …). И приложения для бинарных опционов запрещены.

www.appstorereviewguidelineshistory.com

Cocoa Application Framework (macOS 10.13)

Developer Release Notes. Хотелось бы больше, конечно. Впрочем, использующие AppKit привыкли к такому положению дел. Чтобы проникнуться болью, найдите на странице заголовок 'NSView.canDrawConcurrently' и прочтите единственную строчку под ним.

developer.apple.com
