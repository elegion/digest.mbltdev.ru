Новости

React Native

Свершилось. Наконец-то Facebook представил широкой общественности React Native. Теперь любой желающий может потыкать в него и сделать собственные выводы.

facebook.github.io

ComponentKit

Не менее ожидаемый ComponentKit. Грубо говоря, это реализация React Native без JS, которую инженеры Facebook использовали для ускорения ленты в своем приложении.

github.com

DroneKit

Компания 3D Robotics предлагает способ проще и быстрее разрабатывать приложения для беспилотников. Прямо сейчас поддержка iOS находится в немилом сердцу статусе «coming soon», но к изучению материала можно приступать.

dronekit.io

Уязвимость в AFN 2.5.1

Осторожно используйте эту версию в своих релизах.

blog.mindedsecurity.com

Инструменты

Form

Способ создавать формы. Обратите внимание, в репозитории есть проект с демонстрацией возможностей.

github.com

Paste

Менеджер буфера обмена — вещь старая и скучная. Однако в который раз убеждаюсь, если взять старую проблему и предложить действительно хорошее решение, будет толк. Два разработчика за полгода сделали Paste. Помня о задаче и ресурсах, взгляните на скриншоты с точки зрения программиста. Каково, а?

pasteapp.me

Код

iTC API Docs

Неофициальная документация по iTunes Connect API.

github.com

Тестирование для iOS в 2015 году

Хорошая статья о состоянии дел в вопросе тестирования для iOS на текущий момент. В конце есть список обычных CI.

www.mokacoding.com

CFNetwork Diagnostic Logging

Новый Q&A от Apple, в котором рассказано, как включить «CFNetwork diagnostic logging».

developer.apple.com

nib2objc

Забавный набор утилит, позволяющий сконвертировать NIB-файлы для iOS в Objective-C код. До конца не ясно, когда это может быть полезно, но с академической точки зрения очень занятно.

github.com

Swiftyas

Простой и элегантный способ добавить оператор as в Objective-C. И только почему раньше этого никто не сделал?!

blog.inferis.org

Auto layout best practices for minimum pain

Как выяснилось, многие разработчики не знают о методе -[UIView updateConstraints] и пихают констрейны прямо в layoutSubviews. Если вы один из них, обязательно ознакомьтесь с информацией по ссылке.

medium.com

Core Data vs. ARC

Оказывается, бывает полезно иметь возможность указать время жизни объекта явно.

mjtsai.com

Git from the inside out

Прекрасная статья для тех, кто хочет узнать о том, как GIT работает внутри.

codewords.recurse.com
