Инструменты

GrandPerspective

Небольшая утилита, которая при помощи Tree Map, позволяет быстро оценить, для чего используется драгоценная дисковая память.

grandperspectiv.sourceforge.net

MacDown

Отличный редактор Markdown для OS X. Обязательно попробуйте.

macdown.uranusjr.com

black-screen

Эмулятор терминала. Отличается от стандартного терминала наличием подсказок при наборе и потрясающим UI.

github.com

Код

A WebDriver server for iOS that runs inside the Simulator.

Сервер, который позволяет инструментам для тестирования (например, Selenium) взаимодействовать с вашим приложением (получать иерархию вью, тапать на кнопки, …)

github.com

FBSimulatorControl

Многие рассуждали о том, как было бы здорово запустить несколько симуляторов в одной сессии на OS X. А ребята из FB взяли и сделали. Теперь автоматическое тестирование может выйти на новый уровень.

github.com

tvOS SDK: An iOS Developer’s Initial Impressions

Top Tips for iOS 9 Development
