Инструменты

GitUp

Очень занятный UI для Git.

gitup.co

AnyBar

Небольшой индикатор для меню мака. Цвет индикатора можно менять при помощи отправки UDP пакаета (echo -n "black" | nc -4u -w0 localhost 1738). Можно рассматривать как готовый механизм сигнализации о любых событиях в системе.

github.com

Network Link Conditioner

Удивительно, но не все знают о существовании этой штатной и весьма не новой утилиты для проверки работы вашей программы при разном типе соединения.

blog.crowdint.com

Unclutter

На прошлой неделе мы писали о Paste. Несколько дней назад на глаза попался инструмент с похожей функциональностью и, также, красивым UI.
